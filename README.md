# Student Insight Platform — SMA

Platform manajemen sekolah untuk SMA (Sekolah Menengah Atas).
Produk terpisah dari SIP SMK.

## Stack
- Frontend: Vanilla JS + HTML (GitHub Pages)
- Backend: Supabase (PostgreSQL + RLS + Edge Functions)
- AI: Gemini 2.0 Flash

## Perbedaan dari SIP SMK
- Tidak ada DUDI, PKL, Kaprodi, Program Keahlian
- Struktur akademik: Kelompok Mapel → Mapel → Mapel Pilihan
- Fitur SMA-spesifik: Mapel Pilihan, Dashboard Prestasi
